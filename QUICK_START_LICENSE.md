# Quick Start: Add SurveyJS License

## 3 Simple Steps

### 1️⃣ Add License Key to `.env.local`

```env
NEXT_PUBLIC_SURVEYJS_LICENSE_KEY="YOUR_LICENSE_KEY_HERE"
```

### 2️⃣ Restart Development Server

```bash
# Stop current server (Ctrl+C or Cmd+C)
npm run dev
```

### 3️⃣ Verify in Browser Console

Open DevTools (F12) and check for:
```
✅ SurveyJS Core license activated
✅ SurveyJS Creator license activated
```

---

## That's it! 🎉

The license is now active and will remove all trial watermarks.
